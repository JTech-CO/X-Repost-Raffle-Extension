개요

이 문서는 X Reposts Raffle Extension(이하 “확장 프로그램”)의 개인정보 처리방침입니다. 본 확장 프로그램이 사용자 데이터를 처리하는 경우, 본 방침은 정확하고 최신 정보를 제공합니다.

확장 프로그램은 기본적으로 브라우저 내에서 동작하며, 공개된 리포스트 목록을 화면에서 읽어 추첨을 수행합니다. 서버로 데이터를 전송하지 않습니다.

1. 사용자 데이터 처리 여부

확장 프로그램은 페이지에서 보이는 공개 정보(사용자 아이디/핸들, 게시물 URL·ID 등)를 일시적으로 처리하여 목록 정렬과 추첨을 수행합니다. 또한 사용 편의를 위해 UI 설정(정렬 기준, 키워드 제외, 당첨자 수 등)을 브라우저 로컬 저장소에 보관할 수 있습니다.

2. 포괄적 공개

2-a. 수집·이용·공유 방법

  • 수집: 사용자가 X(트위터) 리포스트 페이지를 열면, 확장 프로그램은 화면의 공개 DOM을 읽어 리포스터 계정을 파싱합니다.
  • 이용: 수집된 공개 데이터는 중복 제거·필터링·추첨 및 결과 내보내기(CSV/JSON 복사)에 사용됩니다.
  • 보관: 추첨 결과와 목록은 메모리에만 존재하며, 설정 값만 로컬 저장소에 저장됩니다. 사용자가 브라우저 데이터를 삭제하면 함께 삭제됩니다.
  • 공유: 기본적으로 제3자와 공유하지 않습니다. 사용자가 내보내기 또는 클립보드 복사를 선택하는 경우, 해당 내보내기 행위는 사용자가 직접 수신자를 결정합니다.

2-b. 공유 대상

기본 기능에서 사용자 데이터를 외부 제3자에게 제공하지 않습니다. 분석/광고 SDK, 원격 서버, 클라우드 전송을 사용하지 않습니다. 단, 사용자가 자발적으로 결과 파일을 전송·게시하는 경우 그 수신자(예: 이벤트 당첨 공지 대상)는 사용자가 지정합니다.

추가 안내

  • 보안: 데이터 처리는 브라우저 환경에 한정됩니다.
  • 아동: 아동을 대상으로 하지 않습니다.
  • 정책 변경: 내용이 변경되는 경우 본 페이지에 갱신일과 함께 게시합니다.

Overview

This Privacy Policy explains how X Reposts Raffle Extension (“the Extension”) handles user data. Whenever the Extension processes user data, this policy provides accurate and up‑to‑date information.

The Extension works locally in your browser. It reads the public retweeters list rendered on the page to run sorting and raffles. No data is sent to any server.

1. Whether We Process User Data

The Extension temporarily processes publicly visible information on the page (e.g., usernames/handles, post URLs/IDs) to sort participants and draw winners. For convenience, UI preferences (sorting, exclude keywords, number of winners, etc.) may be stored in browser local storage.

2. Comprehensive Disclosure

2-a. How We Collect, Use, and Share

  • Collect: When you open an X (Twitter) retweets page, the Extension parses the public DOM to read retweeter accounts.
  • Use: The data is used for de-duplication, filtering, random draw, and exporting results (CSV/JSON copy).
  • Store: Raffle lists/results live in memory only; settings may persist in local storage. Clearing browser data removes them.
  • Share: No automatic sharing. If you export or copy, you decide the recipients of that exported data.

2-b. Parties With Whom Data Is Shared

By default, we do not share user data with any third parties. We do not use analytics/ads SDKs, remote servers, or cloud transfer. If you voluntarily send or publish exported results, the recipients (e.g., announcement audiences) are chosen by you.

Additional Information

  • Security: Processing is confined to the browser environment.
  • Children: Not directed to children.
